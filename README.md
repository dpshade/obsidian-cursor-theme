# Cursor

A clean, comfortable Obsidian theme with a carefully crafted color palette inspired by Nordic design principles.

![Cursor Theme Preview](screenshot.png)

## Features

- **Dual Mode Support**: Fully optimized for both dark and light modes
- **Style Settings Compatible**: Customize colors and typography via the Style Settings plugin
- **Comprehensive Coverage**: Styles all core Obsidian features including:
  - Editor and preview modes
  - Graph view
  - Canvas
  - Callouts
  - Tables
  - Code blocks with syntax highlighting
- **Plugin Compatibility**: Works well with popular plugins like Dataview, Calendar, and Kanban

## Installation

### From Obsidian Community Themes

1. Open **Settings** in Obsidian
2. Navigate to **Appearance**
3. Click **Manage** next to Themes
4. Search for "Cursor"
5. Click **Install and use**

### Manual Installation

1. Download `theme.css` and `manifest.json` from the [latest release](https://github.com/dps/obsidian-cursor-theme/releases)
2. Create a folder called `Cursor` in your vault's `.obsidian/themes/` directory
3. Copy the downloaded files into this folder
4. Open Obsidian Settings > Appearance and select "Cursor"

## Customization

This theme supports the [Style Settings](https://github.com/mgmeyers/obsidian-style-settings) plugin for easy customization.

Available settings:
- **Accent Color**: Change the primary accent color
- **Background Color**: Customize the main background
- **Font Size**: Adjust editor text size
- **Line Height**: Modify line spacing
